# CollegeCubes

College discovery platform to search, compare and predict colleges.

## Features

- College search
- Filters
- College comparison
- College predictor
- Saved colleges
- Dark mode
- Responsive UI

## Tech Stack

Frontend:
- Next.js
- TypeScript
- Tailwind CSS

State:
- Zustand

Data:
- React Query

## Run Locally

npm install

npm run dev
